Comment acheter un produit?

Pour acheter un produit il vous suffit de visiter les pages via la navigation et en cliquant sur la categorie qui vous intéresse. vous pourrez ensuite voir tout le materiel mis en vente ou en troc sur se site et pour cette categorie. Il vous suffira alors de contacter directement le vendeur et vous mettre daccord sur la transaction.

Comment contacter le responsable du site?

Si vous avez la moindre question concernant les articles présenté sur ce site, un probleme rencontré sur l'utilisation d'une page ou n'importe quelle autres questions concernant les sports de glisses ou ce site web. Il vous suffit de cliquer sur le bouton contact présent sur le bandeau inferieur de chaque pages. De remplir les champs requis puis de validez afin que nous recevions votre message pour y répondre dans les plus brefs délai.

Comment crée un compte?

Pour crée un compte rien de plus simple. Sur la premiere page vous trouverez en haut à gauche dans la rubrique compte, un lien (inscripton/connexion). Cliquer dessus pour arriver sur la page correspondante. Enfin vous devriez trouver deux boutons (inscription) et (connexion). Cliquer sur le bouton qui correspond au formulaire rechercher pour faire dérouler le contenu, remplir tout les champs, validez et double cliquer sur le bouton precedement cliquer pour faire remonter le contenu.

Pourquoi souscrire un compte?

L'intérêt de vous inscrire est que vous aurez ensuite un espace de visite dédié et personnaliser pour chaques nouvelle connections. Vous pourez ainsi revoir les produits cliquer comme favorie et beneficier de news personnaliser en fonction de vos selections.

Que faire si le contenu est inapproprié?

Si vous tombez sur une plublication qui n'aurez pas été insérrer dans la bonne catégorie ou qui aurez un contenu déplacé ou inapproprié. Dans ce cas rendez-vous dans la section contact de ce site pour nous envoyer votre requête sans oublier de mentionné la publication ciblé.

Comment vendre son materiel?

Si vous souhaitez vendre un produit, il vous suffit de cliquer sur le bouton (vend ton matos). Ce lien vous renverra vers un formulaire ou il vous suffira de saisir les informations requises. Une fois ce formulaire validez il sera traiter et s'affichera sur la page correspondante à la categorie choisie.

que deviennes mes produit en ventes?

Une fois en ligne, votre materiel reste visible pour une durée de 4 mois, apres quoi il sera supprimer, vous pourrez cepandant le publiez à nouveau.
